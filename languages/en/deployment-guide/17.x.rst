Tuleap 17.5 - March 2026
========================

Test management API update
--------------------------

Changes in endpoint: ``GET /api/testmanagement_campaigns/{id}/testmanagement_executions``

The limit parameter now enforces a maximum value of 50 (previously unlimited). 
Requests trying to call the endpoint with a ``limit`` superior to 50 will now get an HTTP error response with a 400 status.
The pagination header ``X-PAGINATION-LIMIT-MAX`` was already mentioning the limitation, so existing code :ref:`handling collections as expected <dealing-with-collections>` will not be impacted.

Tuleap 17.4 - February 2026
===========================

MySQL 8.4 support and upgrade
-----------------------------

Tuleap supports MySQL 8.4 since this release.
You must plan and upgrade to it before the end of life of MySQL 8.0 in April 2026.

You can report to :ref:`the upgrade guide for more information <mysql84_upgrade>`.
